Dice roller + manual OpenTelemetry

This page is the frontend (static HTML/CSS/JS). It calls the FastAPI backend on /api/*. Each roll creates a span, increments a counter, records a histogram, and emits a structured log — all wired explicitly in Python (no LoggingInstrumentor, no FastAPI auto-instrumentor).
